## notas prova
alert
.serve para exibir uma msg 
.nao recebe resposta bloqueia a tela ate o usuario 
confirma ideal para exemplos simples e aprendizado
. importante o codigo para e espera o usuario

Prompt
.ele ira nos permitir realizar duas açoes 
-mostrar uma pergunta 
-receber uma resposta do usuario
.importante toda resposta do prompt é texto (string)
 mesmo que o usuario digite um numero

prompt + variaveis
. guardar respostas
. o valor digitado pode ser armazenado 
. variaveis passam a ter significado real 
. o codigo começa a reagir ao usuario sem variavel, a resposta se perde
